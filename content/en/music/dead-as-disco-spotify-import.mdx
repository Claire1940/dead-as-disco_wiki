## Understanding the Import Workflow

The game engine requires specific parameters to translate audio files into playable levels. When you navigate to the Stage Select menu and open Infinite Disco, you are presented with an import interface. Keep in mind that the game does not pull directly from streaming services; you must have the audio files saved locally on your machine to initiate the process.

### Essential Calibration Steps

| Step | Action | Purpose |
|---|---|---|
| **1. File Selection** | Navigate to local audio folder | Loads the track into the engine |
| **2. BPM Input** | Enter track-specific BPM | Defines the timing grid |
| **3. Lag Adjustment** | Fine-tune latency slider | Aligns visual hits with audio |
| **4. Test Run** | Play and observe indicators | Validates sync quality |

> **💡 Tip:** If you are unsure of a song's BPM, use a trusted online beat-counter tool or search for the track title followed by "BPM" on Google to find the exact numerical value needed for the game settings.

## Perfecting Your Sync Settings

Once your file is loaded, the most critical factor is the "Lag" parameter. Even if your BPM is set correctly, minor hardware or file-encoding delays can cause your hits to feel "off-beat." The game provides a visual indicator on the side of the screen; if your attacks are landing before or after the beat, you must adjust the lag slider incrementally.

### Troubleshooting Sync Issues

*   **Hits are early:** Increase the lag value slightly to delay the input registration.
*   **Hits are late:** Decrease the lag value to tighten the response time.
*   **Inconsistent feedback:** Ensure no other background audio processing software is running, as this can introduce variable latency.

## Why Local Files Are Required

The reason there is no native **dead as disco spotify import** button is due to DRM (Digital Rights Management) restrictions inherent in streaming platforms. By using local files, the game engine can process the raw audio data without interference, allowing for the precise beat detection required for the rhythm-action mechanics. This ensures that your local custom library remains stable regardless of internet connectivity or service changes.

## FAQ

**Q: Can I use cloud-streamed music files directly in the game?**

A: No, the game requires local audio files (such as .mp3 or .wav) to function correctly. You must download the files to your device to use the import feature.

**Q: How do I find the correct BPM for my music?**

A: You can use online BPM tappers or search for your song title on music databases. Accurate BPM is the foundation of a successful dead as disco spotify import experience.

**Q: Does changing the lag setting permanently affect the song?**

A: The lag setting is saved for each specific track profile. You may need to tweak it once, and the game will remember your preference for that file in the future.

**Q: What should I do if the beat still feels off after calibration?**

A: Double-check your BPM input first. If the BPM is correct but the hits still feel misaligned, perform a "tap-to-beat" calibration again to ensure your own timing isn't the variable causing the delay.
