## Step-by-Step Installation Guide

Installing **dead as disco character mods** requires a manual touch, as the game does not currently feature a built-in Steam Workshop integration. Follow these steps precisely to ensure the skins load correctly during the stage select screen.

### Step 1: Download and Extract
Locate your desired mod on Nexus Mods. For this example, we will use the Spider-Man Advanced Suit. Choose the "Manual Download" option and select "Slow Download" (unless you have a premium account). Once the file is on your PC, right-click the folder and select "Extract All" or "Extract to [Folder Name]." You should see three specific files inside the extracted folder—these are the core assets the game needs to recognize the new skin.

### Step 2: Locate the Game Directory
Open your Steam Library, right-click on *Dead as Disco*, and navigate to **Manage > Browse local files**. This will open the root folder where the game is installed on your hard drive.

### Step 3: Create the Mod Folder
Within the game files, navigate through the following path: `DeadAsDisco > Content > Paks`. Once you are inside the `Paks` folder, you need to create a specialized subfolder. Right-click, select "New Folder," and name it exactly `~mods` (the tilde symbol is crucial for the game engine to prioritize these files).

| Folder Path | Purpose |
| :--- | :--- |
| **.../Content/Paks/** | Original game data storage |
| **.../Content/Paks/~mods/** | Priority folder for custom character skins |

### Step 4: Deploy the Mod Files
Take the three files you extracted in Step 1 and move them directly into the `~mods` folder you just created. Ensure there are no subfolders within `~mods`; the files themselves should reside there for the game to read them properly.

## Testing Your Character Mods

Once the files are in place, launch Dead as Disco through Steam. Navigate to the **Stage Select** menu. If the installation was successful, the character preview for Charlie should now display your chosen mod (e.g., Spider-Man or Hatsune Miku). 

> **💡 Tip:** If the character still looks like Charlie, double-check the spelling of the `~mods` folder. The tilde `~` is often the culprit for failed installations.

## How to Remove or Swap Mods

Managing your collection of **dead as disco character mods** is a straightforward process. Because the game reads everything in the `~mods` folder, "uninstalling" a mod simply means removing its files.

1.  Navigate back to `DeadAsDisco/Content/Paks/~mods`.
2.  Select the files associated with the mod you wish to remove.
3.  Delete the files or move them to a backup folder on your desktop.
4.  Restart the game. Charlie will automatically return to his default appearance.

| Action | Result |
| :--- | :--- |
| **Delete file from ~mods** | Reverts character to default |
| **Add new file to ~mods** | Overwrites default with new skin |
| **Rename ~mods to mods** | Game will likely ignore the custom files |

## Troubleshooting Common Issues

Even with a clear guide, modding can occasionally run into hurdles. Here is a checklist to help you resolve common problems with **dead as disco character mods**:

*   **Game Crashes on Launch**: This usually happens if a mod is outdated or incompatible with the current version of the game. Remove all files from the `~mods` folder and add them back one by one to identify the culprit.
*   **Invisible Character**: If the model doesn't load and the character is invisible, the file path within the `.pak` file might be incorrect. Ensure you are using mods specifically designed for the 2026 build of the game.
*   **T-Posing Models**: This occurs when the skeleton of the modded character doesn't match Charlie’s animations. Check the mod description on Nexus to see if it requires specific animation patches.
