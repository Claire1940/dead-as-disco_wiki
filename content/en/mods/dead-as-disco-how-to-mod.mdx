## Step-by-Step: Dead as Disco How to Mod

The process of installing a skin mod involves creating a specific folder that the game engine looks for when loading assets. Follow these steps carefully to ensure your new characters appear correctly in the stage select menu.

### 1. Downloading Your Chosen Mod
Navigate to the mod page of your choice. For this example, we will focus on the popular "Advanced Suit" skin which replaces Charlie with a certain web-swinging hero. 

*   Click on the **Manual Download** button.
*   Select the **Slow Download** option (unless you have a premium subscription).
*   Save the file to a dedicated folder on your desktop to keep your downloads organized.

### 2. Extracting the Files
Most mods come in a compressed format to save bandwidth. Right-click the downloaded file and select "Extract Here" or "Extract to [Folder Name]." Inside, you will typically find three core files that represent the 3D model, the textures, and the physics data.

### 3. Locating the Game Directory
You need to find where Steam has installed your game. The easiest way to do this is:
1.  Open your **Steam Library**.
2.  Right-click on **Dead as Disco**.
3.  Select **Manage** > **Browse local files**.
4.  Navigate through the folders: `DeadassDisco` -> `Content` -> `Paks`.

### 4. Creating the Mod Folder
Within the `Paks` folder, you must create a new directory specifically for your custom content. The game engine requires a specific naming convention to prioritize these files over the default ones.

> **⚠️ Important:** You must use the tilde symbol (~) at the beginning of the folder name. Name the folder exactly `~mods` to ensure the game loads it correctly.

### 5. Moving the Files
Take the extracted files from your desktop and move them directly into the newly created `~mods` folder. Your file path should look something like this:
`...\SteamLibrary\steamapps\common\Dead as Disco\DeadassDisco\Content\Paks\~mods\`

| File Type | Function |
| :--- | :--- |
| **.pak** | Contains the bulk of the modded data. |
| **.ucas** | Handles asset streaming and indexing. |
| **.utoc** | Provides a table of contents for the game engine. |

## How to Remove or Disable Mods

If you want to return to the vanilla experience or if a mod is causing the game to crash, removing it is a simple process. Since we used a separate folder for our modifications, we don't need to verify game files or reinstall the entire game.

1.  Navigate back to your `~mods` folder in the game directory.
2.  Select the mod files you wish to remove.
3.  **Delete** the files or move them to a backup folder outside of the game directory.
4.  Restart the game.

Once the files are removed from the `~mods` folder, the game will automatically default back to Charlie's original skin. This is the safest way to manage your content without risking a corrupted installation.

*   **Folder Organization**: If you plan on using many mods, you can create sub-folders within `~mods` to keep things tidy. However, ensure that no more than one mod attempts to replace the same character at the same time, as this will lead to visual glitches.

## FAQ

**Q: Can I get banned for using skin mods in Dead as Disco?**

A: Since Dead as Disco is primarily a single-player experience, there is currently no risk of being banned for using cosmetic skin mods. However, always be cautious if the game introduces competitive online leaderboards in the future.

**Q: Why isn't my mod showing up in the stage select menu?**

A: The most common reason is a naming error with the `~mods` folder. Ensure the tilde (~) is at the start of the folder name and that the folder is located inside the `Paks` directory. Also, verify that the files you downloaded are intended for the current version of the game.

**Q: Does learning dead as disco how to mod allow me to add new songs?**

A: While this guide focuses on skin mods, the process for adding custom tracks is similar but often requires additional tools like a chart editor. Check the specific "Stages" or "Music" categories on modding sites for those types of modifications.

**Q: Can I use multiple skins at once?**

A: You can have multiple skin files in your `~mods` folder, but only one can replace a specific character model at a time. For example, you cannot have two different mods both trying to replace Charlie simultaneously without causing a conflict.
