## 단계별 설치 가이드

**dead as disco 스킨 모드**를 설치하려면 수동 작업이 필요합니다. 현재 게임에는 통합된 "Steam 창작마당"이나 인게임 모드 매니저가 없기 때문입니다. 게임이 충돌하지 않고 새로운 스킨이 올바르게 로드되도록 다음 단계를 정확히 따르세요.

### 1. 다운로드 및 압축 해제
스파이더맨 어드밴스드 슈트와 같이 마음에 드는 스킨을 찾았다면 "Manual Download" 옵션을 선택하세요. 대부분의 모드는 `.zip` 또는 `.rar` 파일로 압축되어 있습니다. 사용하기 전에 WinRAR 또는 7-Zip과 같은 도구를 사용하여 이 파일들의 압축을 풀어야 합니다.

### 2. 게임 디렉터리 찾기
시스템에 게임이 설치된 위치를 찾는 방법은 다음과 같습니다.
1. Steam 라이브러리를 엽니다.
2. **Dead as Disco**를 우클릭합니다.
3. **관리** > **로컬 파일 탐색**을 선택합니다.
4. 그러면 게임의 루트 설치 폴더가 포함된 창이 열립니다.

### 3. 모드 폴더 생성
게임 엔진이 커스텀 콘텐츠를 인식하려면 특정 하위 폴더가 필요합니다. 다음 경로를 따라 이동하세요.
`DeadAsDisco > Content > Paks`

`Paks` 폴더 안에 새 폴더를 만들어야 합니다. 언리얼 엔진 모딩에서는 이 폴더 이름 앞에 "물결표" 또는 틸드(~) 접두사를 붙이는 것이 표준 관례입니다. 우클릭하여 새 폴더를 만들고 이름을 다음과 같이 지정합니다.
`~mods`

| 폴더 레벨 | 경로 이름 | 목적 |
| :--- | :--- | :--- |
| **루트** | DeadAsDisco | 메인 게임 디렉터리 |
| **하위 폴더 1** | Content | 핵심 게임 에셋 |
| **하위 폴더 2** | Paks | 압축된 게임 데이터 |
| **대상 폴더** | **~mods** | 모든 `.pak` 스킨 파일의 저장 위치 |

### 4. 파일 이동
다운로드하여 압축을 푼 파일(보통 `.pak`, `.sig`, 또는 `.ucas`로 끝남)을 방금 만든 `~mods` 폴더로 직접 이동합니다.

## 게임 내에서 모드 확인하기

파일을 올바른 디렉터리에 넣은 후 게임을 실행합니다. **dead as disco 스킨 모드**가 작동하는지 확인하려면 다음을 수행하세요.
1. **스테이지 선택(Stage Select)** 화면으로 이동합니다.
2. 캐릭터 미리보기를 확인합니다. 설치가 성공했다면 찰리가 다운로드한 캐릭터 모델(예: 스파이더맨 또는 하츠네 미쿠)로 교체되어 있어야 합니다.
3. 레벨을 시작하여 실제 게임 플레이 중에 애니메이션과 텍스처가 올바르게 로드되는지 확인합니다.

> **⚠️ 경고:** 모드를 추가한 후 게임이 시작 시 충돌한다면 `~mods` 폴더에서 파일을 제거하고 다시 실행해 보세요. 이는 대개 버전 불일치를 의미합니다.

## 스킨 삭제 또는 교체 방법

`~mods` 폴더 방식의 장점 중 하나는 원본 게임 파일을 덮어쓰지 않는다는 것입니다. 모드를 제거하는 방법은 추가한 파일을 삭제하는 것만큼 간단합니다.

1. 다시 `DeadAsDisco/Content/Paks/~mods` 디렉터리로 이동합니다.
2. 더 이상 원하지 않는 스킨과 관련된 파일을 선택합니다.
3. 파일을 삭제하거나 바탕 화면의 백업 폴더로 이동합니다.
4. 게임을 재시작하면 찰리가 기본 모습으로 돌아옵니다.

여러 개의 **dead as disco 스킨 모드** 사이를 전환하려면 한 번에 하나의 캐릭터 교체 모드만 활성화하는 것이 좋습니다. 동일한 "찰리" 모델을 교체하려는 여러 모드가 있으면 시각적 오류나 게임 불안정이 발생할 수 있습니다.
