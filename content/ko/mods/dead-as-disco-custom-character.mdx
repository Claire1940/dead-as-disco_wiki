## 단계별 설치 가이드

**dead as disco 커스텀 캐릭터**를 설치하려면 게임 디렉토리 내에 특정 폴더를 생성해야 합니다. 게임에는 기본적으로 "Mods" 폴더가 없으므로, 엔진이 이러한 커스텀 에셋을 로드하도록 지시하는 특정 명명 규칙을 사용하여 폴더를 직접 만들어야 합니다.

### 1. 선택한 스킨 다운로드
Nexus Mods의 Dead as Disco 섹션으로 이동합니다. 현재 인기 있는 선택지로는 스파이더맨 어드밴스드 슈트, 하츠네 미쿠, 그리고 용과 같이 시리즈의 키류 카즈마 등이 있습니다. 마음에 드는 것을 선택하고 "Manual Download"를 클릭하세요.

### 2. 게임 폴더 찾기
게임이 설치된 위치를 찾으려면 Steam 라이브러리를 열고 Dead as Disco를 마우스 오른쪽 버튼으로 클릭한 다음 **관리**, **로컬 파일 보기**를 차례로 클릭합니다. 그러면 게임의 루트 디렉토리가 열립니다.

### 3. Paks 디렉토리로 이동
루트 폴더에서 다음 경로를 따라갑니다: `DeadAsDisco` > `Content` > `Paks`. 이곳은 게임의 기본 데이터 파일이 저장되는 곳입니다.

### 4. ~mods 폴더 생성
`Paks` 폴더 안에 새 폴더를 만들어야 합니다. 마우스 오른쪽 버튼을 클릭하고 새 폴더를 선택한 다음 이름을 정확히 `~mods`로 지정합니다(물결표 기호가 매우 중요합니다). 이 특정 이름은 언리얼 엔진이 기본 게임 에셋보다 이 폴더 안의 파일을 우선시하도록 지시합니다.

### 5. 파일 추출 및 이동
Nexus Mods에서 다운로드한 `.zip` 파일을 엽니다. 내부에는 일반적으로 세 개의 파일(`.pak`, `.ucas`, `.utoc`로 끝남)이 있습니다. 이 파일들을 복사하여 새로 만든 `~mods` 폴더에 직접 붙여넣습니다.

| 설치 단계 | 필요한 작업 | 핵심 세부 사항 |
| :--- | :--- | :--- |
| **1단계** | Nexus에서 모드 다운로드 | 무료 계정은 "Slow Download" 사용 |
| **2단계** | 로컬 게임 파일 열기 | Steam 관리 설정을 통해 액세스 가능 |
| **3단계** | `Content/Paks`로 이동 | 핵심 데이터 디렉토리 |
| **4단계** | `~mods` 폴더 생성 | **반드시** 물결표(~) 접두사 포함 |
| **5단계** | `.pak` 파일 붙여넣기 | 추출된 파일을 `~mods`로 이동 |

이 모드는 찰리를 Insomniac 게임의 상징적인 슈트를 입은 피터 파커로 교체합니다. 이 슈트는 고해상도 텍스처를 특징으로 하며 Dead as Disco의 네온 빛 환경과 놀라울 정도로 잘 어울립니다. 스파이더맨이 리드미컬한 공격을 수행하는 모습은 게임 플레이에 완전히 다른 맛을 더해줍니다.

### 하츠네 미쿠 (가상 아이돌)
게임의 음악적 특성을 고려할 때 하츠네 미쿠는 자연스럽게 잘 어울립니다. 이 **dead as disco 커스텀 캐릭터** 모드는 세계에서 가장 유명한 보컬로이드를 무대로 불러옵니다. 이 모드에는 종종 긴 트윈테일에 대한 물리 효과가 포함되어 게임의 움직임에 동적으로 반응합니다.

### 키류 카즈마 (용과 같이)
더 "거친" 미학을 원하는 사람들에게는 도지마의 용이 인기 있는 선택입니다. 이 모드는 찰리의 모델을 키류로 교체하여 야쿠자 전설의 냉정한 우아함으로 적들을 쓰러뜨릴 수 있게 해줍니다.

## 모드 삭제 또는 교체 방법

원래의 찰리 캐릭터로 돌아가거나 다른 **dead as disco 커스텀 캐릭터**를 시도하고 싶다면, 과정은 설치만큼이나 간단합니다. 핵심 게임 파일을 수정하는 것이 아니라 `~mods` 폴더에 추가하는 것뿐이므로 게임 설치가 영구적으로 "손상"될 위험은 없습니다.

1.  **게임 종료:** Dead as Disco가 실행 중일 때는 모드 파일을 삭제하지 마세요.
2.  **모드 폴더로 이동:** `DeadAsDisco/Content/Paks/~mods`로 다시 이동합니다.
3.  **파일 삭제:** 제거하려는 모드와 관련된 `.pak`, `.ucas`, `.utoc` 파일을 선택하여 삭제하기만 하면 됩니다.
4.  **확인:** 게임을 실행하고 스테이지 선택 화면으로 이동합니다. 찰리가 원래 모습으로 돌아와 있어야 합니다.

모드를 교체하려면 이전 파일을 삭제하고 다음 캐릭터 선택을 위한 새 파일들을 동일한 `~mods` 디렉토리에 붙여넣으면 됩니다.

## 일반적인 문제 해결

Dead as Disco 모딩은 일반적으로 쉽지만 몇 가지 문제에 부딪힐 수 있습니다. 대부분의 문제는 폴더 이름 지정이나 파일 배치에서 발생합니다.

*   **캐릭터가 나타나지 않음:** 폴더 이름이 처음에 물결표가 있는 `~mods`인지 다시 확인하세요. 그냥 "mods"라고 이름 지으면 게임이 이를 무시합니다.
*   **실행 시 게임 충돌:** 이는 대개 모드가 오래되었거나 파일이 충돌하는 경우에 발생합니다. `~mods` 폴더의 모든 파일을 제거하고 하나씩 다시 추가하여 원인을 찾아보세요.
*   **투명 캐릭터:** 이는 `.ucas` 또는 `.utoc` 파일이 누락된 경우 발생합니다. 모드 다운로드에서 제공된 세 개의 파일이 모두 폴더에 있는지 확인하세요.

| 문제 | 예상 원인 | 해결 방법 |
| :--- | :--- | :--- |
| **모드 무시됨** | 폴더 이름 오류 | 폴더 이름을 **~mods**로 변경 |
| **텍스처 누락** | 불완전한 파일 이동 | 3개의 모드 파일(.pak, .ucas, .utoc)을 모두 이동 |
| **무한 로딩** | 모드 충돌 | 다른 모드를 제거하고 개별적으로 테스트 |

더 많은 공식 업데이트와 커뮤니티 지원을 원하시면 [Dead as Disco Steam 커뮤니티 허브](https://steamcommunity.com/app/2681140)를 방문하여 다른 플레이어들이 무엇을 만들고 공유하는지 확인해 보세요.

## FAQ

**질문: 커스텀 캐릭터 모드를 사용하면 밴(정지)을 당하나요?**

답변: 아니요. Dead as Disco는 싱글 플레이어 경험이며 개발자들은 일반적으로 커뮤니티의 창의성을 지지합니다. **dead as disco 커스텀 캐릭터**를 위해 자신의 게임 파일을 모딩하는 것은 완전히 안전하며 어떠한 밴도 초래하지 않습니다.

**질문: Nexus Mods 외에 다른 스킨은 어디서 찾을 수 있나요?**

답변: Nexus Mods가 주요 허브이지만, 일부 제작자들은 전용 Discord 서버나 X(구 Twitter)와 같은 소셜 미디어 플랫폼을 통해 자신의 작업을 공유합니다. 비공식 소스에서 파일을 다운로드할 때는 항상 주의하고 평판이 좋은 백신 프로그램을 사용하고 있는지 확인하세요.

**질문: 커스텀 캐릭터가 게임 플레이나 히트박스를 변경하나요?**

답변: 대부분의 **dead as disco 커스텀 캐릭터** 모드는 순수하게 외형적인 것입니다. 찰리의 시각적 모델을 교체하지만, 게임 플레이 밸런스가 개발자가 의도한 대로 유지되도록 동일한 히트박스와 이동 속도를 유지합니다.
