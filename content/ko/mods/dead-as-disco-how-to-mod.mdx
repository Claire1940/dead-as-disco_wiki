## 단계별 가이드: Dead as Disco 모딩 방법

스킨 모드를 설치하는 과정에는 게임 엔진이 에셋을 로드할 때 찾는 특정 폴더를 생성하는 작업이 포함됩니다. 새로운 캐릭터가 스테이지 선택 메뉴에 올바르게 나타나도록 다음 단계를 주의 깊게 따르십시오.

### 1. 선택한 모드 다운로드
원하는 모드 페이지로 이동합니다. 이 예시에서는 찰리를 특정 거미 영웅으로 교체하는 인기 있는 "Advanced Suit" 스킨에 집중해 보겠습니다.

*   **Manual Download** (수동 다운로드) 버튼을 클릭합니다.
*   **Slow Download** (느린 다운로드) 옵션을 선택합니다 (프리미엄 구독이 없는 경우).
*   다운로드를 정리된 상태로 유지하기 위해 바탕 화면의 전용 폴더에 파일을 저장합니다.

### 2. 파일 압축 해제
대부분의 모드는 대역폭을 절약하기 위해 압축된 형식으로 제공됩니다. 다운로드한 파일을 마우스 오른쪽 버튼으로 클릭하고 "여기에 압축 풀기" 또는 "[폴더 이름]에 압축 풀기"를 선택합니다. 내부에는 일반적으로 3D 모델, 텍스처 및 물리 데이터를 나타내는 세 개의 핵심 파일이 있습니다.

### 3. 게임 디렉토리 찾기
Steam이 게임을 설치한 위치를 찾아야 합니다. 가장 쉬운 방법은 다음과 같습니다:
1.  **Steam 라이브러리**를 엽니다.
2.  **Dead as Disco**를 마우스 오른쪽 버튼으로 클릭합니다.
3.  **관리** > **로컬 파일 보기**를 선택합니다.
4.  폴더 경로를 따라 이동합니다: `DeadassDisco` -> `Content` -> `Paks`.

### 4. 모드 폴더 생성
`Paks` 폴더 내에 커스텀 콘텐츠를 위한 새 디렉토리를 만들어야 합니다. 게임 엔진은 기본 파일보다 이러한 파일을 우선적으로 처리하기 위해 특정 명명 규칙을 요구합니다.

> **⚠️ 중요:** 폴더 이름의 시작 부분에 물결표(~) 기호를 사용해야 합니다. 게임이 올바르게 로드하도록 폴더 이름을 정확히 `~mods`로 지정하십시오.

### 5. 파일 이동
바탕 화면에서 압축을 푼 파일을 새로 만든 `~mods` 폴더로 직접 이동합니다. 파일 경로는 다음과 같아야 합니다:
`...\SteamLibrary\steamapps\common\Dead as Disco\DeadassDisco\Content\Paks\~mods\`

| 파일 형식 | 기능 |
| :--- | :--- |
| **.pak** | 모딩된 데이터의 대부분을 포함합니다. |
| **.ucas** | 에셋 스트리밍 및 인덱싱을 처리합니다. |
| **.utoc** | 게임 엔진을 위한 목차를 제공합니다. |

## 모드 삭제 또는 비활성화 방법

바닐라(순정) 경험으로 돌아가고 싶거나 모드가 게임 충돌을 일으키는 경우, 제거 과정은 간단합니다. 수정을 위해 별도의 폴더를 사용했으므로 게임 파일을 무결성 검사하거나 게임 전체를 재설치할 필요가 없습니다.

1.  게임 디렉토리의 `~mods` 폴더로 다시 이동합니다.
2.  제거하려는 모드 파일을 선택합니다.
3.  파일을 **삭제**하거나 게임 디렉토리 외부의 백업 폴더로 이동합니다.
4.  게임이 다시 시작합니다.

`~mods` 폴더에서 파일이 제거되면 게임은 자동으로 찰리의 원래 스킨으로 돌아갑니다. 이는 설치 파일 손상 위험 없이 콘텐츠를 관리하는 가장 안전한 방법입니다.

> **💡 팁:** 모드 설치 후 게임이 실행되지 않는 경우, 메인 `Paks` 폴더의 파일을 실수로 삭제하지 않았는지 다시 확인하세요. `~mods` 하위 폴더만 수정되어야 합니다.

## FAQ

**Q: Dead as Disco에서 스킨 모드를 사용하면 밴(정지)을 당할 수 있나요?**

A: Dead as Disco는 기본적으로 싱글 플레이어 게임이므로 현재 코스메틱 스킨 모드 사용으로 인해 밴을 당할 위험은 없습니다. 하지만 향후 경쟁적인 온라인 순위표가 도입될 경우 주의가 필요할 수 있습니다.

**Q: 스테이지 선택 메뉴에 모드가 나타나지 않는 이유는 무엇인가요?**

A: 가장 흔한 원인은 `~mods` 폴더의 명명 오류입니다. 폴더 이름 시작 부분에 물결표(~)가 있는지, 폴더가 `Paks` 디렉토리 안에 있는지 확인하세요. 또한 다운로드한 파일이 현재 게임 버전용인지 확인하십시오.

**Q: dead as disco 모딩 방법을 배우면 새로운 노래를 추가할 수 있나요?**

A: 이 가이드는 스킨 모드에 초점을 맞추고 있지만, 커스텀 트랙을 추가하는 과정도 비슷합니다. 다만 채보 에디터와 같은 추가 도구가 필요한 경우가 많습니다. 모딩 사이트의 "Stages" 또는 "Music" 카테고리를 확인해 보세요.

**Q: 여러 개의 스킨을 동시에 사용할 수 있나요?**

A: `~mods` 폴더에 여러 스킨 파일을 넣을 수는 있지만, 특정 캐릭터 모델 하나당 하나의 모드만 적용할 수 있습니다. 예를 들어, 두 개의 서로 다른 모드가 동시에 찰리를 교체하도록 설정하면 충돌이 발생합니다.
