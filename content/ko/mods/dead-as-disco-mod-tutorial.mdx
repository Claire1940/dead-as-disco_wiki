계정 외에도 PC에 파일 압축 해제 유틸리티가 설치되어 있어야 합니다. 최신 운영 체제는 기본적으로 ZIP 파일을 처리할 수 있지만, 압축된 게임 에셋을 처리할 때는 7-Zip이나 WinRAR과 같은 소프트웨어가 주로 선호됩니다.

## 단계별 Dead as Disco 모드 튜토리얼

모드 설치는 게임의 내부 폴더 내에 특정 디렉토리를 생성하는 구조화된 프로세스입니다. 이 게임은 언리얼 엔진 타이틀에서 흔히 볼 수 있는 "Pak" 시스템을 사용하며, 이는 물결표(~)로 시작하는 폴더에 파일이 있는 경우 엔진이 기본 에셋보다 커스텀 에셋을 우선시하도록 합니다.

### 단계 1: 선택한 모드 다운로드
[Dead as Disco Nexus Mods 페이지](https://www.nexusmods.com/deadassdisco)를 방문하여 사용 가능한 스킨을 찾아보세요. 이 예시에서는 많은 플레이어가 선택하는 스파이더맨 "Advanced Suit" 모드를 선택하겠습니다. "Manual Download" 옵션을 선택하세요. 무료 계정을 사용하는 경우 "Slow Download" 버튼을 누르십시오. 파일 크기가 일반적으로 작기 때문에 몇 초 밖에 걸리지 않습니다.

### 단계 2: 파일 압축 해제
다운로드 디렉토리에서 다운로드한 ZIP 폴더를 찾습니다. 파일을 마우스 오른쪽 버튼으로 클릭하고 "모두 압축 풀기"를 선택하거나 선호하는 압축 해제 도구를 사용하세요. 일반적으로 폴더 내부에는 `.pak` 파일, `.ucas` 파일, `.utoc` 파일이라는 세 가지 고유한 파일이 있습니다. 모드가 올바르게 작동하려면 이 세 가지 구성 요소가 항상 함께 있어야 합니다.

### 단계 3: 게임 디렉토리 찾기
모드를 설치하려면 하드 드라이브에서 게임이 설치된 위치를 찾아야 합니다. Steam을 사용하는 경우 가장 쉬운 방법은 다음과 같습니다.
1. Steam 라이브러리를 엽니다.
2. **Dead as Disco**를 마우스 오른쪽 버튼으로 클릭합니다.
3. **관리** > **로컬 파일 탐색**을 선택합니다.
4. 게임의 루트 디렉토리가 표시된 창이 열립니다.

### 단계 4: 모드 폴더 생성
다음 경로를 따라 이동합니다: `Pagoda` > `Content` > `Paks`. `Paks` 폴더 안에 들어가면 새 하위 폴더를 만들어야 합니다. 마우스 오른쪽 버튼을 클릭하고 "새 폴더"를 선택한 다음 이름을 정확히 `~mods`로 지정합니다. 물결표 기호는 게임 엔진이 이 파일들을 표준 게임 데이터보다 높은 우선순위로 로드하도록 지시하므로 매우 중요합니다.

| 폴더 레벨 | 이름 | 필요한 작업 |
| :--- | :--- | :--- |
| 루트 | Dead as Disco | Steam을 통해 열기 |
| 하위 폴더 1 | Pagoda | 폴더 진입 |
| 하위 폴더 2 | Content | 폴더 진입 |
| 하위 폴더 3 | Paks | **여기에 ~mods 폴더 생성** |

### 단계 5: 설치 완료
단계 2에서 압축을 푼 세 개의 파일을 복사하여 방금 만든 `~mods` 폴더에 직접 붙여넣습니다. 파일 이동이 완료되면 설치가 끝납니다. 이제 게임을 실행하고 스테이지 선택 화면으로 이동하면 찰리 대신 새로운 캐릭터 모델이 적용된 것을 확인할 수 있습니다.

## 모드를 안전하게 관리 및 제거하기

이 모딩 방식의 가장 큰 장점 중 하나는 원래 게임 상태로 되돌리기가 매우 쉽다는 것입니다. 원본 파일을 덮어쓰는 것이 아니라 우선순위 폴더에 새 파일을 배치하는 것이므로, 제거는 간단한 "삭제" 작업만으로 가능합니다.

스킨 모드를 제거하고 기본 찰리 스킨으로 돌아가고 싶다면:
1. `Pagoda/Content/Paks/~mods` 폴더로 다시 이동합니다.
2. 제거하려는 모드와 관련된 파일들을 선택합니다.
3. 파일을 삭제하거나 바탕 화면의 보관 폴더로 이동합니다.
4. 게임을 재시작합니다.

게임은 `~mods` 디렉토리에서 커스텀 파일을 찾을 수 없으면 자동으로 원래 에셋으로 기본 설정됩니다. 이것이 핵심 게임 설치를 손상시키지 않고 캐릭터 스킨을 관리하는 가장 안전한 방법입니다.

## 일반적인 문제 해결

상세한 **dead as disco 모드 튜토리얼**이 있더라도 스킨이 나타나지 않거나 시작 시 게임이 충돌하는 등의 문제가 가끔 발생할 수 있습니다.

*   **물결표 누락:** 모드 폴더 이름이 `~mods`인지 확인하세요. `~` 없이 "mods"로만 명명된 경우 게임이 커스텀 파일을 무시할 수 있습니다.
*   **파일 누락:** 세 가지 파일 유형(.pak, .ucas, .utoc)이 모두 폴더에 있는지 확인하세요. 하나라도 누락되면 모델이 보이지 않거나 "T-포즈" 메쉬로 나타날 수 있습니다.
*   **버전 불일치:** 2026년에 게임에 대규모 업데이트가 적용되면 이전 모드가 작동하지 않을 수 있습니다. Nexus Mods에서 모드의 "최종 업데이트" 날짜를 확인하여 현재 게임 버전과 호환되는지 확인하세요.
*   **모드 충돌:** 둘 다 찰리를 교체하는 두 개의 서로 다른 스킨을 설치하려고 하면 게임이 충돌하거나 그래픽 오류가 발생할 수 있습니다. 한 번에 하나의 캐릭터 교체 모드만 사용하세요.

> **경고:** 메인 `Paks` 폴더 안의 파일을 직접 수정하지 마세요. 기본 게임의 무결성을 유지하기 위해 커스텀 `~mods` 하위 폴더에서만 파일을 추가하거나 제거하십시오.

## FAQ

**Q: 이 dead as disco 모드 튜토리얼은 정식 게임과 데모 모두에 적용되나요?**

A: 네, 2026년 출시 버전의 파일 구조는 일관되게 유지됩니다. `Paks` 디렉토리에 `~mods` 폴더를 만드는 방식은 Dead as Disco의 데모와 정식 버전 모두에서 작동합니다.

**Q: 스킨 모드를 사용하면 밴을 당할 수 있나요?**

A: 일반적으로 아닙니다. Dead as Disco는 주로 싱글 플레이어 경험을 제공합니다. 캐릭터 스킨 모딩은 일반적인 관행이며, 경쟁 환경에서 부정행위를 하거나 불공정한 이득을 얻는 것과는 무관합니다.

**Q: Nexus Mods 외에 다른 모드는 어디에서 찾을 수 있나요?**

A: Nexus가 가장 인기가 많지만, 공식 커뮤니티 Discord나 전문 게임 포럼에서 실험적인 모드를 종종 찾을 수 있습니다. 출처가 불분명한 곳에서 파일을 다운로드할 때는 항상 주의하세요.

**Q: 모드 설치 후 캐릭터가 와이어프레임처럼 보이는 이유는 무엇인가요?**

A: 이는 보통 `.ucas` 또는 `.utoc` 파일이 누락되었을 때 발생합니다. `.pak` 파일뿐만 아니라 압축을 푼 ZIP 폴더의 모든 파일을 `~mods` 디렉토리로 옮겼는지 확인하세요.
