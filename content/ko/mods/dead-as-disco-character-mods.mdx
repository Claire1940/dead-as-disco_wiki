## 설치 준비 사항

**dead as disco 캐릭터 모드**의 세계에 뛰어들기 전에, 반드시 게임 파일을 백업해야 합니다. 모딩은 일반적으로 안전하지만, 단계를 건너뛰거나 폴더 이름을 잘못 지정하면 파일 손상이 발생할 수 있습니다.

2.  **파일 압축 해제 소프트웨어**: 다운로드한 `.zip` 또는 `.7z` 압축 파일을 열기 위해 7-Zip이나 WinRAR 같은 도구가 필요합니다.
3.  **Steam 버전 게임**: 이 가이드는 2026년 Dead as Disco 플레이어들이 가장 많이 사용하는 플랫폼인 Steam의 디렉토리 구조를 기준으로 작성되었습니다.

> **⚠️ 경고:** 악성코드를 방지하기 위해 항상 [Nexus Mods](https://www.nexusmods.com/)와 같은 신뢰할 수 있는 소스에서 모드를 다운로드하세요. 커뮤니티에서 검증되지 않은 제3자 "모드 매니저"에 계정 정보를 절대 공유하지 마세요.

## 단계별 설치 가이드

현재 이 게임은 Steam 창작마당 기능을 내장하고 있지 않기 때문에, **dead as disco 캐릭터 모드**를 설치하려면 수동 작업이 필요합니다. 스테이지 선택 화면에서 스킨이 올바르게 로드되도록 다음 단계를 정확히 따르세요.

### 1단계: 다운로드 및 압축 해제
Nexus Mods에서 원하는 모드를 찾습니다. 이 예시에서는 스파이더맨 어드밴스드 수트(Spider-Man Advanced Suit)를 사용하겠습니다. "Manual Download" 옵션을 선택하고 "Slow Download"를 클릭합니다(프리미엄 계정이 없는 경우). 파일이 PC에 다운로드되면 폴더를 마우스 오른쪽 버튼으로 클릭하고 "모두 압축 풀기" 또는 "[폴더 이름]에 압축 풀기"를 선택합니다. 압축을 푼 폴더 안에 세 개의 특정 파일이 있어야 합니다. 이 파일들이 게임에서 새 스킨을 인식하는 데 필요한 핵심 에셋입니다.

### 2단계: 게임 디렉토리 찾기
Steam 라이브러리를 열고 *Dead as Disco*를 마우스 오른쪽 버튼으로 클릭한 다음, **관리 > 로컬 파일 탐색**으로 이동합니다. 그러면 하드 드라이브에 게임이 설치된 루트 폴더가 열립니다.

### 3단계: 모드 폴더 생성
게임 파일 내에서 다음 경로를 따라 이동합니다: `DeadAsDisco > Content > Paks`. `Paks` 폴더 안에 들어왔다면 전용 하위 폴더를 만들어야 합니다. 마우스 오른쪽 버튼을 클릭하고 "새 폴더"를 선택한 뒤, 이름을 반드시 `~mods`로 지정하세요(게임 엔진이 이 파일들을 우선시하도록 하려면 물결표 `~` 기호가 필수적입니다).

| 폴더 경로 | 용도 |
| :--- | :--- |
| **.../Content/Paks/** | 원본 게임 데이터 저장소 |
| **.../Content/Paks/~mods/** | 커스텀 캐릭터 스킨 우선 폴더 |

### 4단계: 모드 파일 배치
1단계에서 압축을 푼 세 개의 파일을 방금 만든 `~mods` 폴더로 직접 옮깁니다. `~mods` 폴더 안에 하위 폴더가 생기지 않도록 주의하세요. 게임이 파일을 제대로 읽으려면 파일 자체가 해당 폴더에 바로 위치해야 합니다.

## 캐릭터 모드 테스트하기

파일 배치가 끝났다면 Steam을 통해 Dead as Disco를 실행하세요. **스테이지 선택(Stage Select)** 메뉴로 이동합니다. 설치가 성공적이었다면 찰리의 캐릭터 미리보기에 선택한 모드(예: 스파이더맨 또는 하츠네 미쿠)가 표시될 것입니다.

> **💡 팁:** 캐릭터가 여전히 찰리로 보인다면 `~mods` 폴더의 철자를 다시 확인하세요. 물결표 `~`를 빠뜨리는 것이 설치 실패의 가장 흔한 원인입니다.

## 모드 제거 또는 교체 방법

**dead as disco 캐릭터 모드** 컬렉션을 관리하는 방법은 매우 간단합니다. 게임은 `~mods` 폴더에 있는 모든 것을 읽어들이므로, 모드를 "삭제"하려면 해당 파일을 제거하기만 하면 됩니다.

1.  `DeadAsDisco/Content/Paks/~mods` 경로로 다시 이동합니다.
2.  제거하려는 모드와 관련된 파일을 선택합니다.
3.  파일을 삭제하거나 바탕화면의 백업 폴더로 옮깁니다.
4.  게임을 다시 시작합니다. 찰리는 자동으로 기본 외형으로 돌아옵니다.

| 작업 | 결과 |
| :--- | :--- |
| **~mods에서 파일 삭제** | 캐릭터를 기본 상태로 복구 |
| **~mods에 새 파일 추가** | 기본 스킨을 새 스킨으로 덮어씀 |
| **~mods 이름을 mods로 변경** | 게임이 커스텀 파일을 무시할 가능성이 높음 |

## 일반적인 문제 해결

명확한 가이드가 있더라도 모딩 중에 가끔 문제에 부딪힐 수 있습니다. **dead as disco 캐릭터 모드**와 관련된 일반적인 문제를 해결하기 위한 체크리스트입니다.

*   **실행 시 게임 충돌**: 대개 모드가 최신 버전이 아니거나 현재 게임 버전과 호환되지 않을 때 발생합니다. `~mods` 폴더에서 모든 파일을 제거한 다음 하나씩 다시 추가하며 원인이 되는 파일을 찾으세요.
*   **캐릭터가 투명하게 보임**: 모델이 로드되지 않고 캐릭터가 보이지 않는다면 `.pak` 파일 내의 파일 경로가 잘못되었을 수 있습니다. 2026년 빌드용으로 특별히 제작된 모드를 사용하고 있는지 확인하세요.
*   **T-포즈(T-Posing) 모델**: 모드 캐릭터의 스켈레톤(뼈대)이 찰리의 애니메이션과 일치하지 않을 때 발생합니다. Nexus의 모드 설명에서 특정 애니메이션 패치가 필요한지 확인하세요.
