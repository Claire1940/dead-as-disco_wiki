## インフィニット・ディスコ・モードへのアクセス

Dead as Disco におけるカスタムコンテンツへの入り口は「インフィニット・ディスコ（Infinite Disco）」モードです。あらかじめ設定されたトラックと振り付けられた遭遇戦が用意されているストーリーステージとは異なり、インフィニット・ディスコは個人の音楽コレクションのためのサンドボックスとして機能します。まず、メインメニューからステージセレクト画面に移動します。そこから、カスタムファイルブラウザが格納されている「インフィニット・ディスコ」のオプションを探してください。

このメニューに入ると、「Import Song（曲をインポート）」ボタンが表示されます。これをクリックするとファイルナビゲーションウィンドウが開き、ローカルドライブに保存されているオーディオファイルを選択できます。ゲームはさまざまな形式に対応していますが、ビート検出アルゴリズムが十分に機能するように、高品質な .mp3 または .wav ファイルを使用することを一般的にお勧めします。インポートする **Dead as Disco の曲**を選ぶには少し戦略が必要です。アンビエントや複雑すぎるジャズ・フュージョンよりも、明確で力強いパーカッションセクションを持つ曲の方が、ゲームプレイに反映されやすい傾向にあります。

| ステップ | アクション | 結果 |
| :--- | :--- | :--- |
| **1** | ステージセレクトへ移動 | ワールドマップ/レベルリストが開く |
| **2** | **インフィニット・ディスコ**を選択 | カスタムレベルのサンドボックスに入る |
| **3** | **Import Song**をクリック | デバイスのファイルエクスプローラーが開く |
| **4** | オーディオファイルを選択 | トラックがゲームエンジンに読み込まれる |

## 曲のインポートとキャリブレーションのマスター

曲が読み込まれたら、最も重要なステップであるキャリブレーション（調整）が始まります。適切なキャリブレーションが行われていないと、音楽に合わせてキーを叩いているつもりでも、攻撃が鈍く感じられたり「浮いている」ように感じられ、コンボを逃す原因になります。ゲームでは、BPM（1分間あたりの拍数）を調整する主な方法として、手動タップと手動入力の2つが用意されています。

手動タップは、プレビューウィンドウで曲を再生しながら、リズムに合わせてマウスをクリックしたりキーを叩いたりする方法です。直感的ではありますが、自身のタイミングが数ミリ秒ずれるだけで、わずかな不正確さが生じる可能性があります。よりプロフェッショナルな設定を目指すなら、多くのプレイヤーはオンラインデータベースを使用してトラックの正確なBPMを調べることを好みます。「Tokyo Drift」や現代のエレクトロニック・ヒットのような **Dead as Disco のインポート曲**を準備している場合は、「[曲名] BPM」で検索すれば正確な数値が見つかり、それをパラメータフィールドに直接入力できます。

> **💡 プロのヒント：** テンポが変化する曲（プログレッシブ・ロックやライブ録音に多い）では、ゲームがビートを維持するのに苦労することがあります。最高の体験を得るためには、一定のBPMを持つスタジオ録音のトラックを使用することをお勧めします。

## 完璧な同期のためのラグ・パラメーター調整

BPM設定が完璧であっても、視覚的なプロンプトとオーディオが完全に「一致」しないと感じることがあります。これは多くの場合、ハードウェアの遅延やオーディオファイルのエンコード方法が原因です。これを修正するための主要なツールが「ラグ（Lag）」パラメータです。この設定により、ヒットのタイミングをオーディオに対して前後にずらすことができます。

インポートした **Dead as Disco の曲**をテストする際は、画面右側の「Perfect」スコアインジケーターに注意を払ってください。ビートに合わせて叩いているのに「Good」や「Late」判定しか出ない場合は、ラグを調整する必要があります。このプロセスには試行錯誤が必要です。まずは設定をゼロから始め、感覚をテストし、「Perfect」が安定して出るようになるまで10〜20ミリ秒刻みで調整してみてください。

| ラグ設定 | ゲームプレイへの影響 | 推奨されるケース |
| :--- | :--- | :--- |
| **負の値** | ヒットが**早く**表示される | オーディオが視覚より遅れていると感じる場合 |
| **ゼロ（デフォルト）** | 標準的な同期 | すべての新規インポート時の開始点 |
| **正の値** | ヒットが**遅く**表示される | 常に「Early（早い）」判定が出てしまう場合 |

## よくあるインポート問題のトラブルシューティング

すべてのファイルがディスコフロアに最適というわけではありません。BPM設定に関わらず、ヒットが極端にずれるという報告もあります。これは多くの場合、オーディオファイルの冒頭にある「無音のパディング」が原因です。曲が始まる前に3秒間の無音がある場合、ゲームのビートマップはその分だけ正確にずれてしまいます。

これを修正するには、外部のオーディオエディタを使用して無音部分をカットするか、ラグパラメータを使用して遅延を補正します。さらに、ファイルがDRM（著作権管理）で保護されていないことを確認してください。特定のストリーミングサービスの保護されたファイルは、ゲームのインポートツールでは読み取れません。技術的な要件について詳しく知りたい場合は、[公式 Dead as Disco Steam コミュニティ](https://steamcommunity.com/app/1314140)をチェックしてください。そこではプレイヤーが人気曲の事前調整済みBPMリストを共有しています。

## ハイスコアのためのカスタムライブラリ最適化

インフィニット・ディスコで最高ランクを達成するには、**Dead as Disco の曲インポート**機能を競技ツールとして扱う必要があります。高BPMの曲（140-180 BPM）は、一般的に敵の出現頻度が高まり、飛び道具の速度も速くなるため、膨大なスコア倍率を獲得できる可能性があります。しかし、これらのトラックはミスが許される余地がほとんどありません。

初心者の場合は、100〜120 BPMの範囲のトラックから始めてください。これにより、移動メカニクスに慣れるまでの間、視覚的に追いやすい安定したリズムが得られます。上達するにつれて、「スピードコア」や「ドラムンベース」のトラックをインポートして、ゲームエンジンとあなたの反射神経を限界まで追い込んでみましょう。

| ジャンル | 一般的なBPM | 難易度評価 |
| :--- | :--- | :--- |
| **シンセサイザー** | 80 - 110 | 初級 / リラックス |
| **ハウス / ポップ** | 120 - 130 | 中級 / バランス |
| **テクノ / トランス** | 130 - 150 | 上級 / インテンシブ |
| **ドラムンベース / ブレイクコア** | 160+ | エキスパート / カオス |

## オーディオ品質がビート検出に与える影響

ゲームの内部アルゴリズムは、インポートされた **Dead as Disco の曲**の周波数ピークを分析して、どこに「ヒット」を配置するかを決定します。低品質なオーディオファイル（128kbps未満）は、バスドラムとベースギターが混ざり合った「濁った」低音域になることが多く、これが検出システムを混乱させ、不安定なビート配置を招くことがあります。

最良の結果を得るには、320kbpsのビットレートのファイル、またはサポートされている場合は .flac などのロスレス形式を使用してください。トランジェント（ドラムビートの開始時の鋭い立ち上がり）が明確であればあるほど、ゲームはよりクリーンなステージレイアウトを生成できます。もし曲の中に「ゴーストノート（本来ないはずのビート）」が多すぎる場合は、より明瞭度の高いバージョンや、ミックスの異なるバージョンを試してみてください。

> **⚠️ 警告：** 非常に長いトラック（10分以上）をインポートすると、ミドルレンジのハードウェアではメモリの問題やクラッシュが発生することがあります。標準的な3〜5分の曲の長さにとどめるのが最善です。

## FAQ

**Q: BPMを設定した後でも、インポートした曲の同期がずれているように感じるのはなぜですか？**

A: これは通常、「オーディオラグ」または曲の「オフセット」が原因です。BPMが正しくても、曲の最初のビートが正確に0.0秒の地点から始まるとは限りません。インフィニット・ディスコメニューのラグパラメータを調整して、視覚的なヒットを聴覚的なビートに合わせてください。

**Q: 自分のカスタムステージを他のプレイヤーと共有できますか？**

A: 現在、ゲームはオーディオファイルをローカルでインポートします。ステージを共有するには、他のプレイヤーも同じオーディオファイルを所有しており、体験を同一にするためにあなたが使用したのと同じBPMとラグ設定を使用する必要があります。

**Q: インポートできる曲の数に制限はありますか？**

A: インポートできる **Dead as Disco の曲**の数にハードコードされた制限はありませんが、利用可能なディスク容量や、大量のフォルダをインデックス化するゲームの能力がパフォーマンスに影響を与える可能性があります。読み込みを速くするために、カスタムミュージックは専用のフォルダにまとめておくことをお勧めします。

**Q: 間違ったBPMを入力するとどうなりますか？**

A: BPMが高すぎると、ゲームは音楽よりもはるかに速く進行し、リズムを維持することが不可能になります。低すぎると、ゲームプレイが鈍く感じられ、音楽に「遅れている」状態になります。最高の結果を得るために、常に信頼できるオンラインソースでBPMを再確認してください。
