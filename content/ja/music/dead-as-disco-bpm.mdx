## ゲーム内調整ツールの理解

曲を初めてゲームにインポートした際、最初に確認すべきなのはゲーム内の調整メニューです。開発者は、リズムに合わせてタップしたり、エンジンにファイルを分析させたりしてリズムを見つけるための「Calibrate（調整）」ボタンを用意しています。しかし、多くのプレイヤーが犯す一般的な間違いは、このツールによって生成された生の数値だけに頼ってしまうことです。

内部アナライザーは、140.03や179.61といった長い小数点以下の数値を提供することがよくあります。デジタルファイルとしては技術的に正確かもしれませんが、これらの数値は長い曲の中でわずかな「ズレ（ドリフト）」を引き起こす原因になります。最高の体験を得るためには、調整ツールをあくまで開始点として使用し、その後手動で数値を整数に調整する必要があります。

| 機能 | 役割 | 推奨されるアクション |
| :--- | :--- | :--- |
| **Calibrateボタン** | トラックの速度の概算を提供します。 | まずこれを使用して「おおよその数値」を把握します。 |
| **BPM入力フィールド** | ビート値を手動で入力できます。 | 最も近い整数に**切り上げ、または切り捨て**を行います。 |
| **Lag/Offset設定** | ビートマーカーの開始時間を調整します。 | 曲がすぐにビートから始まる場合は **0** に設定します。 |
| **Manual Override** | 自動調整を完全に無視します。 | 複雑なトラックで自動ツールが失敗した場合に使用します。 |

> **💡 ヒント:** BPMやLagを手動で編集するには、数値フィールドをクリックしてバックスペースキーを使用するだけです。多くのプレイヤーは、これらのフィールドがインタラクティブなテキストボックスであることに気づいていません！

## BPMを見つけるための最適な外部ツール

ゲーム内ツールでうまくいかない場合や、リズムを自然に「感じる」ことができない場合は、いくつかの外部ウェブサイトが音楽ファイルに対して非常に正確なデータを提供してくれます。これらは、メタデータがすでに十分に文書化されている有名な楽曲に特に役立ちます。

### 1. Tunebat
Tunebatは、特定のオーディオファイルをアップロードできるため、コミュニティの間で人気があります。BPMを教えてくれるだけでなく、音楽のキー（調）も特定してくれます。これは、将来の2026年のアップデートでキーに基づいたメカニクスが導入された場合に役立つかもしれません。ファイルをアップロードすると、ゲームに直接入力できるきれいな整数が表示されます。

### 2. Song BPM
ファイルが手元にない場合や、ダウンロードする前に曲を確認したい場合は、Song BPMという巨大なデータベースが便利です。アーティスト名とトラック名を入力するだけで、公式のスタジオBPMが表示されます。

| ツール名 | 入力方法 | 最適な用途 |
| :--- | :--- | :--- |
| **Tunebat** | ファイルアップロード | **カスタムリミックス**やマイナーな楽曲。 |
| **Song BPM** | 検索クエリ | **ヒット曲**やメインストリームの音楽。 |
| **Google検索** | 検索クエリ | 素早い確認やコミュニティの「推奨」設定の検証。 |

## 可変BPMの課題

**dead as disco bpm** エンジンについて理解しておくべき最も重要なことの一つは、現時点では「可変BPM」をサポートしていないということです。音楽制作において、一部の曲はAメロとサビで速度が変わったり、劇的な効果を狙って「ブリッジ」部分で大幅にスローダウンしたりすることがあります。

可変BPMの曲をプレイしようとすると、最初の1分間は同期していても、後半になると完全にプレイ不可能になることに気づくでしょう。ゲームでは曲全体に対して一つの静的な値しか設定できないため、一貫した「4つ打ち」のリズムを維持する曲を選ぶ必要があります。

### 最適な曲のタイプ
*   **エレクトロニック・ダンス・ミュージック (EDM):** 通常、厳密なデジタルメトロノームに沿って制作されています。
*   **ポップ・ヒット:** 「California Girls」のような曲は、最初から最後までビートが一貫しているため、コミュニティで完璧な例として挙げられています。
*   **シンセウェーブ:** ゲームの美学に完璧にマッチし、ほとんどの場合、静的なBPMを使用しています。

### 難しい曲のタイプ
*   **ライブ録音のロック/ジャズ:** 人間のドラマーは自然に速度が変動するため、同期させるのが非常に困難です。
*   **プログレッシブ・メタル:** 頻繁な拍子記号やテンポの変化により、ゲームのリズムマーカーが崩れてしまいます。
*   **オーケストラ・サウンドトラック:** DoomのOSTのような曲は象徴的ですが、内部の「パルス」が必ずしもファイルの技術的なBPMと一致せず、真のリズム体験というよりは「ボタン連打」のような感覚になってしまうことがあります。

## 手動調整のステップバイステップガイド

曲が「ズレている」と感じる場合は、以下の手順に従って **dead as disco bpm** 設定を手動で修正してください。このプロセスにより、どんなに扱いにくいトラックでもプレイ可能になります。

1.  **ベースBPMを見つける:** TunebatやSong BPMを使用して公式の数値を取得します。
2.  **入力と丸め:** ゲームに数値を入力し、最も近い整数に丸めます（例：128.04を128に変更）。
3.  **オフセット（Lag）の確認:** ビートが一貫して早い、または遅い場合は、Lag設定を調整します。正の数値はビートを遅らせ、負の数値はビートを早めます。
4.  **サビでテストする:** イントロだけでテストしないでください。曲の中盤までスキップして、ビートマーカーが依然としてドラムを正確に叩いているか確認します。
5.  **最終仕上げ:** 時間の経過とともに曲がドリフトする場合は、その曲には可変BPMが含まれている可能性が高く、ハイスコアを狙うのには適していないかもしれません。

> **警告:** 曲の「ライブ」バージョンは避けてください。常に「スタジオ」または「ラジオエディット」バージョンを探してください。これらは、ゲームが正確に追跡できる固定されたデジタルテンポである可能性がはるかに高いからです。

## 一般的なリズム問題のトラブルシューティング

最高のツールを使っても、ゲームの感覚がしっくりこないことがあります。2026年にプレイヤーが直面する最も一般的な問題とその解決策を以下に示します。

| 問題 | 考えられる原因 | 解決策 |
| :--- | :--- | :--- |
| **ビートがドリフトする** | 可変BPMまたは小数点の誤差。 | **整数**を使用し、スタジオ録音であることを確認してください。 |
| **マーカーが早すぎる** | Lag/Offsetが低すぎる。 | Lagの値を**10刻みで増やして**ください。 |
| **ビートに「感触」がない** | 違う楽器に同期している。 | ボーカルではなく、**メインのバスドラム**にBPMが合っているか確認してください。 |
| **入力が反映されない** | ハードウェアの遅延。 | オーディオ設定の**バッファサイズ**を確認するか、有線ヘッドホンを使用してください。 |

ファイルのインポートに関するより高度なヒントについては、[公式Steamコミュニティハブ](https://steamcommunity.com/app/2791470)でユーザーが作成したリズムプロファイルを確認することをお勧めします。

## FAQ

**Q: なぜゲームは単なる120ではなく、120.003のようなBPMを提示するのですか？**

A: 自動調整ツールはオーディオファイルの正確な長さを分析します。トラックの最初や最後にあるわずかな無音部分が計算を微妙に歪ませることがあります。一貫性を保つためには、ほとんどの場合、手動で意図された整数（120）に変更する方が良い結果を生みます。

**Q: 途中でテンポが変わる曲を直すことはできますか？**

A: 現在、Dead As Discoは一つのトラック内で複数のBPMゾーンを設定することをサポートしていません。曲の速度が変わると、必然的に同期が外れます。最善の方法は、全編を通して一定のテンポを保つ「クラブ・リミックス」や編集版の音源を見つけることです。

**Q: 調整メニューの「Lag」設定は何のためのものですか？**

A: Lag（またはオフセット）は、最初のビートマーカーが表示されるタイミングを決定します。**dead as disco bpm** が正しくても、マーカーがドラムビートの間に一貫して当たっている場合は、Lagを調整して「メトロノーム」の開始位置を音楽の開始位置に合わせる必要があります。

**Q: BPMの高さや低さに制限はありますか？**

A: ゲームは技術的に非常に高いBPMを処理できますが、200を超えるとマーカーが重なり合い、プレイが極めて困難になります。逆に、80 BPMを下回ると、ゲームのコンバットペースに対して遅すぎると感じられるかもしれません。多くのプレイヤーは、110から160 BPMの間が「スイートスポット（最適な範囲）」であると感じています。
