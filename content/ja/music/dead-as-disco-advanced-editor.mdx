## 10月アップデートツールの理解

アドバンスドエディターの導入は、*Dead As Disco* コミュニティにとって重要なマイルストーンです。以前のキャリブレーションシステムは不十分で、デジタルオーディオファイルのわずかな変動に対応できないことがよくありました。新しいツールセットでは、より細かな楽曲マッピングが可能になります。

主な追加機能には、切り替え可能なメトロノーム、洗練されたBPM入力フィールド、そしてダイナミックなビートオフセット調整システムが含まれます。メトロノームは作業を確認するための不可欠なツールですが、ベテランのエディターの多くは、状況に応じて使い分けることを推奨しています。初期のキャリブレーション段階では気が散る原因になることもあるため、まずは基本のリズムを確定させ、その後にメトロノームを起動してトラックの長期的な同期を確認するのが最善です。

## BPMとテンポの安定性を極める

**Dead As Disco アドバンスドエディター**を使用する上で最も重要な点の一つは、テンポを「きれいな」数字にすることです。最初に楽曲をキャリブレーションすると、エディターが114.98や115.02といった小数点以下の多いテンポを提案することがあります。最初の数秒間は正確に見えるかもしれませんが、こうしたわずかな差が3〜4分の曲全体で「ドリフト（ズレ）」を引き起こします。

最良の結果を得るためには、BPMを常に最も近い整数に設定すべきです。キャリブレーションツールが114.97と表示した場合は、手動で115に設定してください。これにより、ゲームの内部クロックが音楽制作で使用される標準的なタイミングと完全に同期した状態を維持できます。

> **💡 プロのヒント：** 画面上のノートに対して曲が徐々に速くなったり遅くなったりするように感じる場合、BPMがわずかにズレている可能性があります。長期的な同期ズレを防ぐため、常に整数を使用するようにしましょう。

## ビートオフセットの調整方法

完璧なBPMを設定しても、最初のビートがオーディオと一致していなければ、違和感が生じます。ここでビートオフセットの出番です。**Dead As Disco アドバンスドエディター**でオフセットを調整するには、初心者にはすぐには分かりにくい特定の操作が必要です。

ビートの位置を操作するには、**Shift**キーと**マウスホイール**（中央ボタン）を同時に押し続ける必要があります。これらを保持したままマウスを左右に動かすと、オーディオ波形の上でビートグリッド全体がスライドします。これにより、垂直のビートマーカーを音楽ファイル内の「トランジェント」（音の立ち上がりの鋭い部分）に合わせることができます。

| ステップ | 操作 | 視覚・聴覚的な合図 |
| :--- | :--- | :--- |
| **1** | BPMを設定する | グリッドが曲の全体的な速度と一致していることを確認。 |
| **2** | 最初のビートを特定する | 波形の中から最初の主要なドラムヒットやシンセ音を見つける。 |
| **3** | Shift + クリック | Shiftとマウス中央ボタンを押し続けてグリッドを掴む。 |
| **4** | 整列させる | 最初のマーカーがオーディオのピークに乗るまでグリッドをドラッグする。 |
| **5** | テストする | メトロノームをオンにして曲を再生し、ズレがないか確認する。 |

## 視覚的キャリブレーション vs 聴覚的キャリブレーション

**Dead As Disco アドバンスドエディター**を使用する際、完璧な同期を見つけるための主な方法は2つあります。耳を使う方法と、目を使う方法です。プロのマッパーの多くは、これらを組み合わせて使用しています。

### 聴覚的キャリブレーション（ドラム法）
テンポを設定する最も簡単な方法は、パーカッションを聴くことです。現代の音楽の多くは、キックドラムやスネアで「ダウンビート」を定義しています。これらの一定のヒット音に集中することで、リズムに合わせてタップしたり、メトロノームを使用してクリック音がドラムの打点と正確に一致するかを確認したりできます。ドラムがない曲の場合は、脈打つようなベースラインや鋭いボーカルの立ち上がりなど、他のリズム要素を探してください。

### 視覚的キャリブレーション（波形法）
視覚的なアプローチを好むなら、波形表示が最大の味方になります。オーディオ内の「ピーク」を探してください。これらは通常、ビートが始まるトラックの最も音量が大きい部分です。大きくズームインすることで、音の波が立ち上がる正確な位置を確認できます。開始時間をこれらの特定のピークに合わせることで、人間の反応速度に伴う推測を排除できます。

> **⚠️ 注意：** 一部のオーディオファイルには、冒頭にわずかな「無音」が含まれています。エディターで開始時間を調整してこれを考慮しないと、マップ全体が遅れてしまいます。

## カスタム楽曲の高度なトラブルシューティング

基本の手順に従っても、曲が「ぼやけて」感じられることがあります。これは多くの場合、オーディオファイルのエンコード方法や音楽のジャンルに起因します。例えば、可変ビットレート（VBR）ファイルは、リズムゲームのエンジンで問題を引き起こすことがあります。

**Dead As Disco アドバンスドエディター**内で問題が解決しない場合は、以下のチェックリストを参考にしてください。

| 問題 | 考えられる原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **早い段階でのズレ** | 不正確なBPM | 曲が実際には120ではなく120.5 BPMではないか確認する。 |
| **後半でのズレ** | 小数点を含むBPM | BPMを整数に変更し、オフセットを再調整する。 |
| **入力遅延** | ハードウェアのレイテンシ | ゲームのメインメニューでオーディオバッファ設定を確認する。 |
| **カクつき** | ファイル形式 | オーディオを320kbpsの標準的な .Ogg または .Mp3 に変換する。 |

技術的なサポートやコミュニティ製のマップについては、[Official Dead As Disco Steam Community](https://steamcommunity.com/app/dead-as-disco) を訪れて、作品を共有したり他のエディターからフィードバックを得たりすることができます。

## カスタムトラックの仕上げ

ビートを整列させテンポを設定したら、最後のステップはフルプレイテストです。少なくとも2回はプレイすることを強くお勧めします。1回目はメトロノームをオンにして技術的な正確さを確認し、2回目はオフにして曲の「フィール（感覚）」が正しいかを確認するためです。**Dead As Disco アドバンスドエディター**は強力な味方ですが、カスタムトラックで「公式レベル」の完成度を実現するには、多少の試行錯誤が必要です。

エディターが許す限りズームインすることを恐れないでください。リズムゲームにおいて精度は命であり、わずか数ミリ秒のズレが、ゲームプレイ中の「Perfect」と「Great」判定の分かれ目になります。忍耐強く、2026年のアップデートで提供されたツールを活用すれば、どんな曲でも *Dead As Disco* の傑作に変えることができます。

## FAQ

**Q: なぜ曲の始まりは完璧なのに、最後の方でリズムがズレてしまうのですか？**

A: これはほとんどの場合、BPMが整数でないことが原因です。テンポが120.04のように設定されていると、その余分な0.04ビートが数分間で蓄積されます。**Dead As Disco アドバンスドエディター**を使用してBPMを正確に120に設定し、開始オフセットを再調整してください。

**Q: BPMを変えずにビートグリッドを移動するにはどうすればよいですか？**

A: Shiftキーとマウスホイール（中央クリック）を押し下げたまま、マウスを左右にドラッグします。これにより、ビート自体の速度を変えることなく、グリッド全体（オフセット）を移動できます。

**Q: メトロノームの音が大きすぎて音楽が聞こえません。音量を変えられますか？**

A: エディター内でメトロノームの音量を個別に変更することはできませんが、多くのプレイヤーはメトロノームを「オフ」にしてオフセットを調整し、確認の時だけ「オン」にする方法が最もやりやすいと感じています。

**Q: Dead As Disco アドバンスドエディターはすべてのオーディオ形式をサポートしていますか？**

A: MP3やWAVなどの一般的な形式のほとんどをサポートしていますが、ゲームエンジン内での最高の安定性とパフォーマンスを得るためには、高品質な .Ogg ファイルの使用が一般的に推奨されます。
