## リズムベースの戦闘システムのマスター

Dead As Discoの核心は、人気シリーズ『バットマン：アーカム』のフリーフロー・メカニズムを採用し、それを一定のリズムに同期させた戦闘システムにあります。ダメージを最大化し、フローを維持するには、すべてのパンチ、キック、カウンターをトラックに合わせて繰り出す必要があります。

### 基本的な戦闘アクション

成功の鍵は、音楽との同期を助けるオプションの視覚補助「ビートマーカー」を使いこなすことです。成功の尺度は「フィーバーメーター」で、コンボを途切れさせずに攻撃や回避を成功させることで蓄積されます。

| アクション | 入力タイミング | 効果 |
| :--- | :--- | :--- |
| **通常攻撃** | ビートに合わせる | 標準的なダメージを与え、フィーバーを溜める。 |
| **カウンター** | ビートに合わせる（フラッシュ時） | 敵の攻撃を跳ね返し、フローを維持する。 |
| **回避** | ビートに合わせる | ガード不能攻撃を避け、チャーリーの立ち位置を変える。 |
| **フィーバーテイクダウン** | メーター最大時 | 通常の敵に対する即死級のシネマティックなフィニッシュムーブ。 |

### 高度なスペシャリストの敵

ネオン輝く街を進むにつれ、単なる雇われの悪党以上の敵に遭遇することになります。特定のパターンの戦略を必要とするスペシャリストタイプが登場します。

1.  **ロボット・レディース:** 長距離レーザーを発射します。最初のチャージを回避し、4拍目の最後のパルスに合わせてカウンターを当てる必要があります。
2.  **ライオットシールド・ヘビー:** 正面からの攻撃が効きません。リズムに合わせた「ヴォルト（跳び越し）」ムーブで背後に回り込み、露出した背中を攻撃する必要があります。
3.  **ソニック・スナイパー:** リズムを乱すオフビートの弾を発射します。コンボを維持するために、これらのターゲットを優先的に排除してください。

## カスタマイズとアンコール・ハブ

ミッションの合間に、チャーリーは拠点となるダイブバー「**アンコール**」に戻ります。このハブはキャラクターの成長や外見のカスタマイズに不可欠です。早期アクセス期間中、開発者はこのスペースでのインタラクションの拡張に取り組んでいます。

*   **スキルツリー:** 戦闘で集めた「ファン」を使い、ラピッドストライクや範囲衝撃波などの新しい技をアンロックします。
*   **コスメティック:** 好みのディスコスタイルに合わせてチャーリーの衣装を変更できます。
*   **会場の改善:** リソースを消費してアンコール・バーを修理します。2026年現在のビルドでは一部の装飾変更にバグがある可能性がありますが、今後のパッチで修正される予定です。

## インフィニット・ディスコとカスタム音楽サポート

開発者が言及した最も期待される機能の一つが「**インフィニット・ディスコ・モード**」です。このサバイバルモードでは、プレイヤーは次第に難易度が上がる敵の波に耐え抜くことになります。特筆すべきは、自分の音楽トラックをアップロードして設定できる機能です。

お気に入りのモダンなシンセサイザー・トラックや名曲に合わせて悪党の群れと戦うことを想像してみてください。ゲームエンジンがアップロードされたファイルのBPMを解析してビートマップを生成するため、無限のコンテンツが提供されます。この機能だけでも、Steam版の価値は非常に高く、発売時に **dead as disco game pass** 版がないことへの不満も和らぐでしょう。
