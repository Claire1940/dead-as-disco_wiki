## ステップ・バイ・ステップ：Dead as DiscoのMod導入方法

スキンModのインストールプロセスでは、ゲームエンジンがアセットをロードする際に参照する特定のフォルダを作成する必要があります。新しいキャラクターがステージ選択メニューに正しく表示されるよう、以下の手順を慎重に進めてください。

### 1. 選択したModのダウンロード
お好みのModページに移動します。この例では、チャーリーを某ウェブスイングヒーローに置き換える人気の「Advanced Suit」スキンに焦点を当てます。

*   **Manual Download**（マニュアルダウンロード）ボタンをクリックします。
*   **Slow Download**（スローダウンロード）オプションを選択します（プレミアム会員でない場合）。
*   ダウンロードしたファイルを整理しやすくするため、デスクトップの専用フォルダに保存します。

### 2. ファイルの展開（解凍）
ほとんどのModは帯域幅を節約するために圧縮形式で提供されます。ダウンロードしたファイルを右クリックし、「ここに展開」または「[フォルダ名]に展開」を選択します。中には通常、3Dモデル、テクスチャ、物理データを表す3つのコアファイルが含まれています。

### 3. ゲームディレクトリの特定
Steamがゲームをインストールした場所を見つける必要があります。最も簡単な方法は以下の通りです：
1.  **Steamライブラリ**を開きます。
2.  **Dead as Disco**を右クリックします。
3.  **管理** > **ローカルファイルを閲覧**を選択します。
4.  フォルダを辿ります：`DeadassDisco` -> `Content` -> `Paks`

### 4. Modフォルダの作成
`Paks`フォルダの中に、カスタムコンテンツ専用の新しいディレクトリを作成する必要があります。ゲームエンジンがデフォルトのファイルよりもこれらのファイルを優先して読み込むためには、特定の命名規則を使用する必要があります。

> **⚠️ 重要:** フォルダ名の先頭には必ずチルダ記号（~）を使用してください。ゲームが正しくロードできるように、フォルダ名を正確に `~mods` としてください。

### 5. ファイルの移動
デスクトップに展開したファイルを、新しく作成した `~mods` フォルダに直接移動します。ファイルパスは以下のようになります：
`...\SteamLibrary\steamapps\common\Dead as Disco\DeadassDisco\Content\Paks\~mods\`

| ファイル形式 | 機能 |
| :--- | :--- |
| **.pak** | Modデータの大部分が含まれています。 |
| **.ucas** | アセットのストリーミングとインデックス作成を処理します。 |
| **.utoc** | ゲームエンジン用の目次（テーブルオブコンテンツ）を提供します。 |

## Modの削除または無効化方法

バニラ（デフォルト）の体験に戻したい場合や、Modが原因でゲームがクラッシュする場合は、削除プロセスは簡単です。改造用に別のフォルダを使用したため、ゲームファイルの整合性確認やゲーム全体の再インストールは必要ありません。

1.  ゲームディレクトリ内の `~mods` フォルダに戻ります。
2.  削除したいModファイルを選択します。
3.  ファイルを**削除**するか、ゲームディレクトリ以外のバックアップフォルダに移動します。
4.  ゲームを再起動します。

`~mods` フォルダからファイルが削除されると、ゲームは自動的にチャーリーのオリジナルスキンに戻ります。これが、インストールを破損させるリスクなくコンテンツを管理する最も安全な方法です。

## 上級者向けのMod導入ヒント

**Dead as DiscoのMod導入方法**のスキルを次のレベルに引き上げたい場合は、コミュニティの最も活発なメンバーが使用している以下のプロのヒントを考慮してください：

*   **フォルダの整理**: 多くのModを使用する予定がある場合は、`~mods` 内にサブフォルダを作成して整理整頓できます。ただし、複数のModが同時に同じキャラクターを置き換えようとしないように注意してください。視覚的なグリッチの原因になります。
*   **アップデートの確認**: ゲームに公式パッチが適用されると、Modが機能しなくなることがよくあります。お気に入りのスキンのNexus Modsページを常にチェックし、2026年ビルド用のアップデート版がリリースされていないか確認してください。
*   **セーブデータのバックアップ**: スキンModは一般的に安全ですが、ゲームファイルをいじる前にセーブデータをバックアップしておくのが常に良い習慣です。

> **💡 ヒント:** Modインストール後にゲームが起動しない場合は、メインの `Paks` フォルダ内のファイルを誤って削除していないか再確認してください。変更を加えるのは `~mods` サブフォルダのみにしてください。

## よくある質問（FAQ）

**Q: Dead as DiscoでスキンModを使用するとBANされますか？**

A: Dead as Discoは主にシングルプレイヤー体験であるため、現在、外見変更用のスキンModを使用してBANされるリスクはありません。ただし、将来的にオンラインランキングなどの競争要素が導入される場合は、注意が必要です。

**Q: ステージ選択メニューにModが表示されないのはなぜですか？**

A: 最も一般的な原因は、`~mods` フォルダの命名ミスです。フォルダ名の先頭にチルダ（~）が付いているか、そしてそのフォルダが `Paks` ディレクトリ内にあるかを確認してください。また、ダウンロードしたファイルが現在のゲームバージョンに対応しているかも確認してください。

**Q: Dead as DiscoのMod導入方法を学べば、新しい曲を追加できますか？**

A: このガイドはスキンModに焦点を当てていますが、カスタム楽曲を追加するプロセスも似ています。ただし、譜面エディタなどの追加ツールが必要になることが多いです。それらの種類の改造については、Modサイトの「Stages」または「Music」カテゴリを確認してください。

**Q: 一度に複数のスキンを使用できますか？**

A: `~mods` フォルダに複数のスキンファイルを入れることは可能ですが、特定のキャラクターモデルを置き換えられるのは一度に1つだけです。例えば、競合を起こさずに2つの異なるModで同時にチャーリーを置き換えることはできません。
