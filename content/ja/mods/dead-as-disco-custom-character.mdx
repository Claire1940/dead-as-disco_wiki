## Dead as DiscoのMOD導入を始める前に

チャーリーを新しいヒーローに変身させる前に、環境を整える必要があります。Dead as DiscoのMOD導入は、標準的なUnreal Engineのファイル構造に従っているため比較的簡単ですが、ゲームが新しいファイルを認識するようにするためにいくつかの特定のステップが必要です。

まず、MODの信頼できる入手先が必要です。コミュニティでは主にNexus Modsを使用してスキン交換やステージ編集をホストしています。「Manual Download（手動ダウンロード）」機能を利用するために、無料アカウントを作成することを強くお勧めします。また、カスタムキャラクターファイルのほとんどは圧縮された`.zip`や`.rar`形式で配布されているため、7-ZipやWinRARなどのファイル解凍ソフトがインストールされていることを確認してください。

| 要件 | 目的 | 推奨ツール |
| :--- | :--- | :--- |
| **Nexus Modsアカウント** | コミュニティ製スキンの閲覧・ダウンロード | 無料アカウント |
| **解凍ソフト** | 圧縮されたMODフォルダの展開 | 7-Zip または WinRAR |
| **ファイルエクスプローラー** | ゲームのローカルインストールフォルダへのアクセス | Windows Explorer |

> **警告:** MODをインストールする前に、必ずセーブファイルのバックアップを取ってください。スキン交換は一般的に安全ですが、予期せぬゲームのアップデートによって、MODファイルが存在するとクラッシュを引き起こす場合があります。

## ステップバイステップ・インストールガイド

**Dead as Discoのカスタムキャラクター**をインストールするには、ゲームディレクトリ内に特定のフォルダを作成する必要があります。ゲームにはデフォルトで「Mods」フォルダが含まれていないため、エンジンにこれらのカスタムアセットを読み込ませるための命名規則に従って作成する必要があります。

### 1. 好きなスキンをダウンロードする
Nexus ModsのDead as Discoセクションにアクセスします。現在人気があるのは、「スパイダーマン・アドバンスドスーツ」、初音ミク、そして『龍が如く』シリーズの桐生一馬などです。お気に入りを選び、「Manual Download」を選択してください。

### 2. ゲームフォルダを探す
ゲームがどこにインストールされているかを確認するには、Steamライブラリを開き、Dead as Discoを右クリックして「管理」を選択し、「ローカルファイルを閲覧」をクリックします。これでゲームのルートディレクトリが開きます。

### 3. Paksディレクトリへ移動する
ルートフォルダから、`DeadAsDisco` > `Content` > `Paks` の順にフォルダを開きます。ここがゲームの主要なデータファイルが保存されている場所です。

### 4. ~modsフォルダを作成する
`Paks`フォルダ内に新しいフォルダを作成する必要があります。右クリックして「新規作成」＞「フォルダ」を選択し、名前を正確に `~mods` としてください（チルダ記号「~」が非常に重要です）。この特定の名前により、Unreal Engineはデフォルトのゲームアセットよりもこのフォルダ内のファイルを優先的に読み込むようになります。

### 5. ファイルを解凍して移動する
Nexus Modsからダウンロードした`.zip`ファイルを解凍します。通常、中には3つのファイル（`.pak`、`.ucas`、`.utoc`で終わるもの）が入っています。これらのファイルをコピーし、作成した`~mods`フォルダに直接貼り付けてください。

| インストール手順 | 必要なアクション | 詳細 |
| :--- | :--- | :--- |
| **手順 1** | NexusからMODをダウンロード | 無料アカウントは「Slow Download」を使用 |
| **手順 2** | ローカルゲームファイルを開く | Steamの「管理」設定からアクセス可能 |
| **手順 3** | `Content/Paks`へ移動 | コアデータディレクトリ |
| **手順 4** | `~mods`フォルダを作成 | **必ず**チルダ(~)プレフィックスを含めること |
| **手順 5** | `.pak`ファイルを貼り付け | 解凍したファイルを`~mods`へ移動 |

利用可能なMODの種類は日々増えています。このゲームは滑らかなアニメーションとリズミカルな動きが特徴であるため、特定のキャラクターをゲームに置き換えると非常に印象的です。以下は、2026年時点でダウンロード可能な評価の高いスキンの一部です。

## MODの削除・入れ替え方法

元のチャーリーに戻したい場合や、別の**Dead as Discoカスタムキャラクター**を試したい場合、そのプロセスはインストールと同じくらい簡単です。コアゲームファイルを変更しているわけではなく（`~mods`フォルダに追加しているだけなので）、ゲームインストールを完全に「壊す」リスクはありません。

1.  **ゲームを閉じる:** Dead as Discoが実行されている間は、MODファイルを削除しないでください。
2.  **MODフォルダへ移動:** `DeadAsDisco/Content/Paks/~mods` に戻ります。
3.  **ファイルを削除:** 削除したいMODに関連付けられている `.pak`、`.ucas`、`.utoc` ファイルを選択して削除するだけです。
4.  **確認:** ゲームを起動し、ステージ選択画面に移動します。チャーリーが元の姿に戻っているはずです。

MODを入れ替えるには、古いファイルを削除し、次のキャラクターの新しいファイルを同じ`~mods`ディレクトリに貼り付けるだけです。

## トラブルシューティング

Dead as DiscoのMOD導入は一般的に簡単ですが、いくつかの問題に直面することがあります。ほとんどの問題は、フォルダ名やファイルの配置ミスに起因します。

*   **キャラクターが表示されない:** フォルダ名が先頭にチルダを付けた `~mods` になっているか再確認してください。単に「mods」となっている場合、ゲームは無視します。
*   **起動時にゲームがクラッシュする:** これは通常、MODが古いか、競合するファイルがある場合に発生します。`~mods`フォルダからすべてのファイルを一度取り出し、原因を特定するために1つずつ戻してみてください。
*   **キャラクターが透明になる:** `.ucas` または `.utoc` ファイルが欠落している場合に発生します。MODのダウンロードに含まれる3つのファイルすべてがフォルダ内に存在することを確認してください。

| 問題 | 考えられる原因 | 解決策 |
| :--- | :--- | :--- |
| **MODが無視される** | フォルダ名の間違い | フォルダ名を **~mods** に変更 |
| **テクスチャが表示されない** | ファイル移動の不完全さ | 3つのファイル（.pak, .ucas, .utoc）すべてを移動 |
| **無限ロード** | MODの競合 | 他のMODを削除し、個別でテストする |

公式のアップデートやコミュニティサポートについては、[Dead as Disco Steam コミュニティハブ](https://steamcommunity.com/app/2681140)にアクセスして、他のプレイヤーが作成・共有している内容を確認してください。

## FAQ

**Q: カスタムキャラクターMODを使用するとBANされますか？**

A: いいえ。Dead as Discoはシングルプレイヤー体験であり、開発者は一般的にコミュニティの創造性を支持しています。**Dead as Discoカスタムキャラクター**のために自分のゲームファイルをMOD化することは完全に安全であり、BANされることはありません。

**Q: コンソール版のゲームでもこれらのMODを使用できますか？**

A: 現在、スキンMODはSteamやその他のPCストアを通じたPC版でのみ利用可能です。PlayStation 5やXbox Series Xなどのコンソールでは、これらのインストールに必要なファイルディレクトリへのアクセスが許可されていません。

**Q: Nexus Mods以外でスキンを見つけるには？**

A: Nexus Modsが主要なハブですが、一部のクリエイターは専門のDiscordサーバーやX（旧Twitter）などのソーシャルメディアプラットフォームで作品を共有しています。非公式のソースからファイルをダウンロードする場合は常に注意し、信頼できるウイルス対策ソフトを使用していることを確認してください。

**Q: カスタムキャラクターはゲームプレイや当たり判定を変更しますか？**

A: ほとんどの**Dead as Discoカスタムキャラクター**MODは純粋に外見的なものです。チャーリーの視覚モデルを置き換えますが、ゲームプレイのバランスが開発者の意図通りに維持されるよう、同じ当たり判定と移動速度を保持します。
